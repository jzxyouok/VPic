# [VPic](https://licao404.github.io/VPic/)

> A sharp and light photo editor based on Vue.js 2.0.

### Info
VPic 是一款基于 Vue.js 2.0 的在线图像处理工具，使用了ElementUI等开源库，希望实现日常场景中图片的快速处理：
例如图像裁切，滤镜添加。
